# Ansible Collection - wzzrd.molecule_test

Sole intention of this simple repo is to test molecule in devspaces. 

It contains:
  - vscode configuration (theme and some settings)
  - vscode extensions
  - a simple collection for testing
  - a default molecule scenario
  - the definion (Containerfile) to bring ansible tooling into DevSpaces
  - some scripts related to the above 

Simply test by running `molecule test --command-borders --report`
